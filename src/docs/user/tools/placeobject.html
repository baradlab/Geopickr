Place Object

Place Object visualises the spatial distribution of subtomograms within a tomogram. A geometric object (or a custom surface) is translated and rotated according to the position and orientation stored for each particle in a TOM/AV3 motive list, and displayed in the same coordinate space as the tomogram. This is a ChimeraX port of the UCSF Chimera plugin described in Qu K. et al. (2018) PNAS 115:E11751–E11760.

Open the tool from Tools → Volume Data → Geopickr and use its Place Object tab (or run the command described below). Each motive list is shown as a single ChimeraX surface model whose particles are drawn as instances, so even lists with many thousands of particles display efficiently.

Motive lists

Click Open motive list... to read a .em file. The list at the top shows every loaded motive list; selecting one shows its controls below. Close removes the selected model.

A motive list is a (20, N) matrix following the TOM/AV3 convention. The rows used by Place Object are:

RowMeaning
1Cross-correlation coefficient (CCC)
8–10X/Y/Z coordinate in the full tomogram
11–13X/Y/Z shift within the subvolume
17–19Phi / Psi / Theta Euler angles (ZXZ, degrees)
20Class number

Object

  • Shape – a built-in shape (Sphere, ArrowX/Y/Z, Rectangle, Triangle, Square, Pentagon, Hexagon) or Custom STL... to browse for your own surface. A custom object can be exported from a density map with Save → STL after centering and orienting it.
  • Voxel size – scales the object to match the pixel size of the coordinates.
  • Z offset – shifts each object along its own (rotated) Z axis.
  • Phi offset – adds a rotation about Z before placement.

Colour

  • By class – colour particles by the class value in the chosen Class row (default row 20), cycling through a fixed palette.
  • By cross-correlation – colour on a ramp between two colours across the chosen CC range. Auto resets the range to the data min/max.
  • Solid – a single colour for all particles.

Show

  • All / None – show or hide every particle.
  • Single – show one particle, chosen by index.
  • CC range – show particles whose CCC falls between the two thresholds.
  • Class – show only the chosen class.

Actions

  • Focus – frame the view on the displayed objects.
  • Save shown... – write the currently displayed particles to a new .em motive list.

Command

placeobject  file  [shape shape-name] [voxelSize s] [zOffset z] [phiOffset deg]

Opens the motive list file, places the chosen object at every particle, and opens the Place Object tool. Example:

placeobject ~/data/particles.em shape ArrowZ voxelSize 4.4
