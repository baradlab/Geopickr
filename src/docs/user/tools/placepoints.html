Place Points

Place Points is a thin helper around ChimeraX's built-in marker tools for the first step of geometric particle picking (the equivalent of the old Volume Tracer plugin). Use it to mark the centers of spherical features or the axes of tubes/filaments in a tomogram, then hand the markers to the Geometry Picker.

Suggested workflow

  1. Open a tomogram and show a slice (Volume Viewer / plane display).
  2. Click New marker set (one set per tube/filament; spheres can share a set).
  3. Choose a placement mode, then right-click in the view to add markers:
    • Mark on plane – drop a marker on the displayed volume plane.
    • Mark on surface – drop a marker on a surface/density.
    • Mark free point – drop a marker at the front clip plane.
  4. Optionally Save .cmm… for reuse, then open the Geometry Picker.

Marker radius sets the display size of new markers. All marker placement, editing and linking use ChimeraX's native Markers tools, so anything you can do there (move, delete, link) works on these marker sets too.
